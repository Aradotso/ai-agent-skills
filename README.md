# AI Agent Skills

Auto-generated agent skills from the fastest-growing **AI agents** open source projects on GitHub, by [ara.so](https://ara.so).

A bot scans GitHub every 30 minutes for trending repos in the AI agents niche, ranks them by stars/day, and writes an installable `SKILL.md` for the top new one.

## Install all skills

```bash
npx skills add Aradotso/ai-agent-skills
```

## Install a specific skill

```bash
npx skills add Aradotso/ai-agent-skills --skill <skill-name>
```
